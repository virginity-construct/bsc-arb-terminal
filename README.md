# BSC DEX Arbitrage Terminal

A real-time web interface for tracking arbitrage opportunities across major DEXes on Binance Smart Chain (BSC).

## Features

- Real-time price tracking across multiple DEXes (PancakeSwap, Apeswap, Biswap)
- Live arbitrage percentage calculations
- Interactive chart visualization
- Auto-refreshing data (10-second intervals)
- Error handling and status indicators
- Responsive terminal-style design

## Setup

1. Clone this repository
2. Open `index.html` in a modern web browser
3. The terminal will automatically start tracking arbitrage opportunities

## Supported Tokens

- BNB (Wrapped BNB)
- CAKE (PancakeSwap Token)

## Technical Details

- Uses Web3.js for blockchain interaction
- Chart.js for data visualization
- Axios for API requests
- Pure JavaScript for real-time updates
- CSS animations for live indicators
